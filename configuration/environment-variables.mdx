This page documents all environment variables used to configure the AgrospAI Data Space Portal. These are defined in the `.env` file at the root of your project.

## Required Variables

### Wallet and Blockchain Connection

<ParamField path="NEXT_PUBLIC_INFURA_PROJECT_ID" type="string" required>
  Your Infura project ID for blockchain connectivity. Used as a fallback RPC provider.
  
  **Default:** `xxx`
  
  ```bash
  NEXT_PUBLIC_INFURA_PROJECT_ID="your_infura_project_id"
  ```
</ParamField>

<ParamField path="NEXT_PUBLIC_WALLETCONNECT_PROJECT_ID" type="string" required>
  WalletConnect project ID for wallet connection functionality.
  
  ```bash
  NEXT_PUBLIC_WALLETCONNECT_PROJECT_ID="your_walletconnect_project_id"
  ```
</ParamField>

### Marketplace Fee Configuration

<ParamField path="NEXT_PUBLIC_MARKET_FEE_ADDRESS" type="string" required>
  The Ethereum address where marketplace fees will be sent.
  
  **Default:** `0x9984b2453eC7D99a73A5B3a46Da81f197B753C8d`
  
  ```bash
  NEXT_PUBLIC_MARKET_FEE_ADDRESS="0x9984b2453eC7D99a73A5B3a46Da81f197B753C8d"
  ```
</ParamField>

<ParamField path="NEXT_PUBLIC_PUBLISHER_MARKET_ORDER_FEE" type="string">
  Publisher market fee taken upon ordering an asset (absolute value). Declared on ERC20 creation.
  
  **Default:** `0`
  
  ```bash
  NEXT_PUBLIC_PUBLISHER_MARKET_ORDER_FEE="1"
  ```
</ParamField>

<ParamField path="NEXT_PUBLIC_PUBLISHER_MARKET_FIXED_SWAP_FEE" type="string">
  Fee received by the publisher market when a datatoken is bought from a fixed rate exchange (percentage).
  
  **Default:** `0`
  
  ```bash
  NEXT_PUBLIC_PUBLISHER_MARKET_FIXED_SWAP_FEE="1"
  ```
</ParamField>

<ParamField path="NEXT_PUBLIC_CONSUME_MARKET_ORDER_FEE" type="string">
  Consume market fee taken upon ordering an asset (absolute value). Specified on order.
  
  **Default:** `0`
  
  ```bash
  NEXT_PUBLIC_CONSUME_MARKET_ORDER_FEE="1"
  ```
</ParamField>

<ParamField path="NEXT_PUBLIC_CONSUME_MARKET_FIXED_SWAP_FEE" type="string">
  Fee received by the consume market when a datatoken is bought from a fixed rate exchange (percentage).
  
  **Default:** `0`
  
  ```bash
  NEXT_PUBLIC_CONSUME_MARKET_FIXED_SWAP_FEE="1"
  ```
</ParamField>

## Advanced Settings

### Pricing Options

<ParamField path="NEXT_PUBLIC_ALLOW_FIXED_PRICING" type="boolean">
  Enable or disable fixed pricing option during price creation.
  
  **Default:** `true`
  
  ```bash
  NEXT_PUBLIC_ALLOW_FIXED_PRICING="true"
  ```
</ParamField>

<ParamField path="NEXT_PUBLIC_ALLOW_FREE_PRICING" type="boolean">
  Enable or disable free pricing option during price creation.
  
  **Default:** `true`
  
  ```bash
  NEXT_PUBLIC_ALLOW_FREE_PRICING="true"
  ```
</ParamField>

<ParamField path="NEXT_PUBLIC_ALLOW_DYNAMIC_PRICING" type="boolean">
  Enable or disable dynamic pricing option during price creation.
  
  **Default:** `false`
  
  ```bash
  NEXT_PUBLIC_ALLOW_DYNAMIC_PRICING="false"
  ```
</ParamField>

### Privacy and Compliance

<ParamField path="NEXT_PUBLIC_PRIVACY_PREFERENCE_CENTER" type="boolean">
  Enable GDPR-compliant privacy preference center to manage cookies. When enabled, uses `gdpr.json` file in the content directory.
  
  **Default:** `false`
  
  ```bash
  NEXT_PUBLIC_PRIVACY_PREFERENCE_CENTER="true"
  ```
</ParamField>

<ParamField path="NEXT_PUBLIC_COMPLIANCE_URI" type="string">
  URI for compliance information.
  
  **Default:** `https://www.delta-dao.com/compliance`
  
  ```bash
  NEXT_PUBLIC_COMPLIANCE_URI="https://www.delta-dao.com/compliance"
  ```
</ParamField>

<ParamField path="NEXT_PUBLIC_COMPLIANCE_API_VERSION" type="string">
  Version of the compliance API to use.
  
  **Default:** `2210`
  
  ```bash
  NEXT_PUBLIC_COMPLIANCE_API_VERSION="2210"
  ```
</ParamField>

## Example Configuration

Here's a complete example `.env` file for production:

```bash
# Wallet and Blockchain
NEXT_PUBLIC_INFURA_PROJECT_ID="your_infura_project_id"
NEXT_PUBLIC_WALLETCONNECT_PROJECT_ID="your_walletconnect_project_id"

# Marketplace Fees
NEXT_PUBLIC_MARKET_FEE_ADDRESS="0x9984b2453eC7D99a73A5B3a46Da81f197B753C8d"
NEXT_PUBLIC_PUBLISHER_MARKET_ORDER_FEE="1"
NEXT_PUBLIC_PUBLISHER_MARKET_FIXED_SWAP_FEE="1"
NEXT_PUBLIC_CONSUME_MARKET_ORDER_FEE="1"
NEXT_PUBLIC_CONSUME_MARKET_FIXED_SWAP_FEE="1"

# Pricing Options
NEXT_PUBLIC_ALLOW_FIXED_PRICING="true"
NEXT_PUBLIC_ALLOW_FREE_PRICING="true"
NEXT_PUBLIC_ALLOW_DYNAMIC_PRICING="false"

# Privacy
NEXT_PUBLIC_PRIVACY_PREFERENCE_CENTER="true"

# External Services
AGROPORTAL_API_KEY="your_agroportal_api_key_here"

# Telemetry
NEXT_TELEMETRY_DISABLED=1
```
