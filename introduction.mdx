# Welcome to AgrospAI Data Space Portal

The AgrospAI Data Space Portal is a next-generation Web3 marketplace designed specifically for the agricultural sector, enabling secure and decentralized data exchange. Built on Ocean Protocol and integrated with the Gaia-X ecosystem, it provides a trusted environment for publishing, discovering, and accessing agricultural datasets and AI algorithms.

<Note>
The portal is part of the Pontus-X initiative, a Gaia-X Web3 Ecosystem that ensures European data sovereignty and compliance.
</Note>

## What is AgrospAI Data Space?

AgrospAI Data Space is a specialized data marketplace that combines the power of blockchain technology with agricultural data management. It enables:

- **Data Publishers** to monetize their agricultural datasets securely
- **Data Consumers** to access high-quality agricultural data and AI models
- **Researchers** to collaborate and share insights within a compliant framework
- **Developers** to build innovative agricultural solutions using trusted data

## Key Concepts

<AccordionGroup>
  <Accordion title="Datatokens">
    Datatokens are ERC-20 tokens that represent access rights to a dataset or service. Each asset has its own unique datatoken. To access a dataset, users must acquire the corresponding datatoken either by purchasing it or receiving it from the publisher.
  </Accordion>
  
  <Accordion title="NFTs (Non-Fungible Tokens)">
    Each published asset is represented by an NFT (ERC-721 token) that establishes ownership and provenance. The NFT holder has control over the asset's metadata and pricing settings.
  </Accordion>
  
  <Accordion title="DDO (Decentralized Data Object)">
    The DDO is a JSON document containing all metadata about an asset, including its description, services, pricing, and access control information. It follows the Ocean Protocol specification and is stored encrypted on-chain.
  </Accordion>
  
  <Accordion title="DID (Decentralized Identifier)">
    Each asset has a unique DID (e.g., `did:op:abc123...`) that serves as its permanent identifier across the Ocean Protocol network. DIDs are blockchain-agnostic and resolve to the asset's DDO.
  </Accordion>
</AccordionGroup>
